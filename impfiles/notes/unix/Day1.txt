Features of Unix:
1.Multi User - More than 1 user will be able to use it simultaenously 
2. Multi Tasking - Multiple programs can run at the same time 
3.Multi Process - execute several process simultaenously
4.Open source - free source code 
5.Highly secured 
6.Portability - allows user to transfer data from 1 system to another 
7.Programming facility - shell can be used for scripting language 
8.Piping

Basic Commands:
ls - display the content (dir and files)
ls -a -> gives all including hidden files
ls -l -> long listing 

Working with directories:
mkdir dir_name 
cd dir_name 
pwd - pressent working dir
cd .. - navigate to previous dir 
rmdir dir_name - delete the dir only when it is empty 
rm -r dir_name - delete the dir when files are present 

Working with files:
touch filename - create file only 
cat - create, write,read,append 
cat > filename - if file is already existing, it will be over written 
if not existing, it will be created 
cat < filename / cat filename 
cat >> filename 
rm filename - to remove the file 

to close : 
ctrl+ D / ctrl + C

vi - create,write and edit 
vi filename 
click on insert button 
esc + :wq - write and quit 
to read the content again 
cat filename

Date command:
print the current date in predefined format 
date +"%options"
Req: August/16-2022

%a- weekday name - Sun 
%A - Sunday 
%b - month name - Jan 
%B - January 
%c - both date and time 
%C - century 
%d - day of the month -01....31 
%m - month -01...12
%y - last 2 digits of the year - 98,99
%Y -1998,1999,2000
%H - hours in 24hr format 00....23
%I - hours in 12 hr format 00....12
%M - minutes 00....59
%S -seconds 
%x - same as %d:%m:%y
%T - same as %H:%M:%S
