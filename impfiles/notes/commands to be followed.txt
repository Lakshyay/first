How to set an environment variable for your account:
     File explorer ->This PC ->Local disk C ->Users ->Username ->Appdata ->Roaming ->npm
*Note: If you don't find Appdata ensure that you have selected hidden items which is present in view.
     copy the npm path.

Search for Edit environmental variables for your account ->double click on the path ->click on New ->Paste the path which you have copied -> click ok ->Again click ok.

Now your path is set.
